Padma Advanced Description

== Description ==

Padma Advanced offers a growing selection of Blocks to extend Padma Unlimited Theme Builder functionalities.

This plugin is in the early stages of development, so we appreciate your comments and reports.

FREE:
ACCORDION BLOCK
The Accordion block allows creating blocks with hidden content – spoilers (toggles). Hidden content will be shown when the block title is clicked. Specify different icons or even use different styles for each spoiler. Included in Visual Elements plugin.
 
BASIC HEADING
A Heading can act as a title, section heading, and/or subheading. You can give each Heading a relative level of importance, from H1 to H6. Tip: Search engines (and people!) use Headings to determine the most important themes and topics of your content.
 
BOX BLOCK
The box block allows creating boxes with colorful titles.Easily change box appearance, place HTML code or shortcodes. Included in Visual Elements plugin.
 
BUTTON BLOCK
The Button Block allows creating highly customizable buttons through the Visual Editor. Modify the button style, colors, and size; add an icon or description with ease. Included in Visual Elements plugin.
 
COLUMNS BLOCK
The Columns block allows organizing page content into columns. Included in Visual Elements plugin.
 
CONTENT TO ACCORDION BLOCK
The Content to Accordion block allows creating blocks with hidden posts Content.  Hidden content will be shown as the post title is clicked.  Each hidden content (Spoiler) can have different icons or use different styles.
 
Content to Cards Block
The Content to Cards block allows displaying expandable Contents by category or content type. 
 
CONTENT TO TABS BLOCK
The content to Tabs block allows organizing content into horizontal or vertical tabs.Specify which tab will be selected by default and turn any tab into a link. 
 
 
DAILYMOTION BLOCK
The Dailymotion block allows inserting responsive Dailymotion videos. Included in Visual
 
DIVIDER BLOCK
The Divider block allows dividing page content with a styled divider.Customize colors, hide the “Got to top” link and adjust divider size.
 
DUMMY IMAGE BLOCK
The Dummy image block allows displaying a dummy image.Image theme and size can be modified.
 
DUMMY TEXT BLOCK
The Dummy text block allows you to display “lorem ipsum” text for a fast and easy layout preview.  Just select the number of paragraphs or words required and it would be automatically generated.
 
 
PRO:
FONTAWESOME BLOCK
The Fontawesome block allows using Font Awesome icons in the Visual Editor. Change the size, color and other style options with the Design Mode. 
 
 
GMAP BLOCK
The Gmap block allows displaying and easily setting up Google Maps in the Visual Editor
 
HEADING BLOCK
The Heading block allows creating styled headings with customizable size and margin.
 
LABEL BLOCK
The Label block creates and customizes unlimited labels.
 
LIGHTBOX BLOCK
The Lightbox block allows displaying several elements (images, web pages, or HTML content) in a pop-up window.
 
PORTFOLIO
Allows you to create blocks with hidden posts content. Hidden content will be shown when the block title will be clicked. You can specify different icons or even use different styles for each spoiler.
 
PORTFOLIO CARDS
Allows you to create blocks with hidden posts content. Hidden content will be shown when block title will be clicked. You can specify different icons or even use different styles for each spoiler.
 
POST DATA BLOCK
The Post Data block allows displaying and stylizing several post fields like post title, post content, modified date and more.
 
SPACER BLOCK
The Spacer block adds empty space between elements in a layout.
 
SPOILER BLOCK
The Spoiler block allows creating blocks with hidden content – spoilers (toggles). Hidden content will be shown when the block title is clicked.Specify different icons or even different styles for each Spoiler.
 
TABS BLOCK
The Tabs block allows splitting content into horizontal or vertical tabs. Specify which tab will be selected by default or turn any tab into a link. HTML code or shortcodes can be used as content.
 
VIMEO BLOCK
The Vimeo block is a responsive block that allows inserting Vimeo videos.
 
YOUTUBE BLOCK
The Youtube block allows inserting responsive YouTube videos. Create playlists using Youtube Advanced block.
 
QUOTE BLOCK
The Quote block allows inserting “quotes” into the content. Quote, author and link can be specified.
